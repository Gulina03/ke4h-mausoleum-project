Mausoleum of Khoja Ahmed Yasawi

Knowledge Graph Enrichment Project for Knowledge Engineering for the Humanities

Project Overview

This project investigates the representation of the Mausoleum of Khoja Ahmed Yasawi in Wikidata. Using SPARQL queries, we explored existing information, identified knowledge gaps, and used Large Language Models (ChatGPT and Gemini) to propose RDF-based knowledge graph enrichment.

Research Question

What cultural, religious, and historical information about the Mausoleum of Khoja Ahmed Yasawi is missing from Wikidata, and how can it be added using Large Language Models?

SPARQL Queries

  • Query 1: Explore existing information in Wikidata.
  • Query 2: Retrieve unique heritage designations using DISTINCT.
  • Query 3: Check architect information using OPTIONAL.
  • Query 4: Filter results using FILTER.
  • Query 5: Organize metadata using ORDER BY and LIMIT.

Knowledge Gaps

  1. Connection with Yasawiyya Sufi Order
  2. Role as a Pilgrimage Site
  3. Spiritual and Religious Significance
  4. Connections with Related Religious Heritage Sites in Kazakhstan

LLM Evaluation

ChatGPT and Gemini were evaluated using Zero-shot, Few-shot, and Chain-of-Thought prompting techniques.

ChatGPT produced focused outputs aligned with the research questions and identified knowledge gaps. Gemini generated richer and more exploratory semantic enrichment proposals.

Conclusion

The project explored Wikidata using SPARQL queries, identified missing knowledge, evaluated ChatGPT and Gemini, and proposed RDF triples to enrich the representation of the Mausoleum of Khoja Ahmed Yasawi.
